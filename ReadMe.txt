Syobon Action X is currently under active development.
Please report any bugs or suggestions at the GameJolt page and at GitHub Issue Tracker.

Controls:
Game:
Up - Jump
Left,Right - Move
Insert - switch to Level Editor

Editor:
T - next tile
R - previous tile
0 - reset view
1 - blocks
2 - backgrounds
3 - "bonuses"
F1 - main menu

Music used:
Controlled Chaos - no percussion Kevin MacLeod (incompetech.com)
