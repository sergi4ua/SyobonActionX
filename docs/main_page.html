Welcome to Syobon Action X!
This documentation file was designed to guide you through the level editor.
As of this point of time the game is still in active development. So stay tuned for updates! :)

. Key shortcuts:
F1 - Main Menu
T - next tile/entity (switch beetween tiles/entities)
R - previous tile/entity
Left Mouse Button - place object
Right Mouse Button - remove object (eraser mode)
P - set player spawn
0 - Reset view
1 - Default ground tiles
2 - Backgrounds (tree, hills, etc.)
3 - Bonus objects and checkpoint
4 - Enemies
.

The Main Menu


New Level - start a new level (without saving)
Load level... - load a level
Save level... - save current level
Fullscreen (F4) - go into fullscreen
Help - brings up help
Level Completion Options... - what to do when player complete the level.
Message Block message... - what text to display when player hits a message block ( )
Protect level - protect the level from opening in the editor (irreversible)
More... - more level options
Return to main menu - return to main menu
.

Limits

Maximum objects: ~4078
Maximum width of the level: 64000
.

Menu Option: More...


Set Music... - allows you to set custom music for your level
Set Background... - switch the background for your level (along with the tileset)
